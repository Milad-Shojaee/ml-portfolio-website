ICU Sepsis Prediction

This project develops a machine learning pipeline for predicting sepsis risk 6 hours before clinical onset using ICU time-series data.

Problem

Sepsis is a life-threatening condition where early detection is critical. The goal is to use patient measurements such as heart rate, temperature, blood pressure, oxygen saturation, and lab values to predict whether a patient is at risk of developing sepsis within the next 6 hours.

Dataset

I used ICU patient data from PhysioNet, containing multivariate clinical time-series measurements collected during hospital stays.

Method

  • Cleaned missing clinical measurements
  • Engineered temporal features from rolling windows
  • Created a 6-hour prediction horizon
  • Trained baseline logistic regression and XGBoost models
  • Evaluated performance using ROC-AUC, PR-AUC, and F1-score

Results

The tuned XGBoost model achieved strong predictive performance, with ROC-AUC around 0.89 and PR-AUC around 0.48, outperforming the baseline model.

Tools

Python, Pandas, NumPy, Scikit-learn, XGBoost, Matplotlib, PhysioNet

Future Deployment

This model could be deployed as a clinical dashboard where doctors enter or stream patient measurements and receive an early warning risk score.

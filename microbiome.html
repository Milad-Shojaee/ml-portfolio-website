Microbiome Disease Risk Modeling

This project focuses on predicting disease risk from gut microbiome profiles using machine learning and explainable AI. The goal was to identify microbial features associated with chronic fatigue syndrome and improve disease-risk prediction from high-dimensional biological data.

Project Overview

  • Domain: Biomedical AI and microbiome analysis
  • Data type: Gut microbiome species-abundance profiles
  • Task: Disease risk prediction
  • Model: XGBoost classifier
  • Interpretability method: SHAP feature attribution

Methods

I developed a machine learning pipeline to classify disease risk from microbiome abundance profiles. The workflow included preprocessing microbial features, training gradient-boosted decision tree models, evaluating predictive performance, and interpreting key bacterial species contributing to model decisions.

  • Processed high-dimensional gut microbiome feature profiles.
  • Trained XGBoost models for disease-risk classification.
  • Compared model performance against baseline methods.
  • Used SHAP values to identify microbial features driving predictions.
  • Interpreted model outputs in the context of biomedical disease risk.

Results

  • Improved ROC-AUC by 9% compared with baseline approaches.
  • Identified key bacterial species associated with disease-risk prediction.
  • Demonstrated the usefulness of explainable machine learning for microbiome-based risk modeling.

Key Skills Demonstrated

  • Biomedical machine learning
  • Microbiome data analysis
  • XGBoost modeling
  • Explainable AI
  • SHAP feature interpretation
  • Model evaluation using ROC-AUC

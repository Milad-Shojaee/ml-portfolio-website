Power System Fault Diagnosis Using Deep Learning

This project focused on developing intelligent fault diagnosis systems for electrical transmission networks using deep learning and transfer learning techniques. The objective was to automatically identify fault types from voltage and current measurements while maintaining high reliability across varying transmission line configurations.

Project Overview

  • Domain: Power Systems and Smart Grid Analytics
  • Task: Multi-class fault classification
  • Input Signals: Three-phase voltage and current measurements
  • Classes: 10 transmission line fault types
  • Framework: Deep Learning and Transfer Learning

Methods

I designed a deep learning framework capable of identifying transmission line faults directly from electrical measurements. To improve scalability and reduce retraining requirements, I also developed a transfer learning strategy that adapts pre-trained models to new transmission line configurations with minimal additional training.

  • Developed deep neural network models for fault classification.
  • Learned discriminative representations from voltage and current waveforms.
  • Implemented transfer learning across different transmission line lengths.
  • Reduced training time while maintaining diagnostic performance.
  • Evaluated robustness under diverse operating conditions.

Results

  • Achieved 99.5% fault classification accuracy.
  • Successfully classified 10 transmission line fault categories.
  • Reduced training time by approximately 50% using transfer learning.
  • Maintained high diagnostic reliability across different transmission line configurations.

Key Skills Demonstrated

  • Deep Learning
  • Transfer Learning
  • Signal Processing
  • Power Systems
  • Time-Series Classification
  • PyTorch
  • Feature Learning

This work demonstrates how machine learning can improve the reliability and efficiency of modern power systems by enabling fast and accurate automated fault diagnosis.

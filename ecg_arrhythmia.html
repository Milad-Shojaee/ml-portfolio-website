ECG Arrhythmia Detection

Patient-independent abnormal heartbeat detection using ECG morphology, RR interval dynamics, and targeted augmentation

Project Overview

This project develops a deep learning system for patient-independent ECG abnormal heartbeat detection using the MIT-BIH Arrhythmia Database. The model combines heartbeat morphology from ECG waveforms with RR interval timing features to classify heartbeats as normal or abnormal.

Unlike a random beat-level split, this project uses a DS1/DS2-style inter-patient evaluation protocol, where the model is trained and tested on ECG records from different patients. This provides a more realistic assessment of generalization to unseen subjects.

Motivation

Arrhythmia detection is not only a waveform classification problem. Beat timing and rhythm dynamics are also important. This project systematically investigates how ECG morphology and RR interval features complement each other for abnormal heartbeat detection.

The project also explores the effect of class imbalance and targeted minority-class augmentation for improving abnormal beat detection under patient-independent evaluation.

Methods

  • Dataset: MIT-BIH Arrhythmia Database
  • Task: Binary heartbeat classification: normal vs. abnormal
  • Input 1: ECG beat window centered around annotated heartbeat locations
  • Input 2: RR interval features: previous RR and next RR intervals
  • Model: Multi-input neural network with a 1D CNN ECG branch and dense RR branch
  • Evaluation: Patient-independent DS1/DS2 split
  • Robustness strategy: Minority-class ECG augmentation applied only during training
  • Reporting: Mean and standard deviation over five random seeds

Model Architecture

The ECG branch learns beat morphology using one-dimensional convolutional layers. The RR branch learns rhythm information from neighboring beat intervals. The ECG and RR representations are concatenated and passed to a final classifier. 

ECG beat window  ──► 1D CNN ──► ECG morphology features
RR intervals     ──► Dense NN ─► Rhythm timing features

ECG features + RR features ──► Fusion classifier ──► Normal / Abnormal

Key Results

Model Accuracy ROC-AUC F1-score
ECG morphology only 91.94 ± 2.20% 0.902 ± 0.014 0.617 ± 0.049
RR intervals only 92.87 ± 0.78% 0.818 ± 0.020 0.647 ± 0.008
ECG + RR intervals 94.61 ± 0.52% 0.965 ± 0.004 0.742 ± 0.017
ECG + RR + Augmentation 95.39 ± 0.60% 0.968 ± 0.007 0.772 ± 0.024

The fused ECG + RR model outperformed both morphology-only and rhythm-only baselines, demonstrating that heartbeat morphology and rhythm timing provide complementary information. Targeted augmentation of abnormal beats further improved average accuracy, ROC-AUC, and F1-score.

Ablation Study

Several training strategies were compared to understand the effect of feature fusion and imbalance handling. Weighted binary cross-entropy improved abnormal-beat recall but increased false positives and reduced overall performance. Naive oversampling showed signs of overfitting, while targeted ECG augmentation improved the effective diversity of the minority class.

Clinical and Technical Relevance

This work demonstrates a near-real-time abnormal beat detection pipeline. In a streaming ECG system, incoming data could be processed through R-peak detection, beat extraction, RR interval calculation, and model inference to flag normal or abnormal beats.

Because the model uses both ECG morphology and rhythm timing, it is better suited for abnormal beat screening than a morphology-only classifier.

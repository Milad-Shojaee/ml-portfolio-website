Crop Disease Diagnosis Platform

An end-to-end deep learning system for diagnosing crop diseases from plant leaf images using ResNet-18, Vision Transformer, PyTorch, Streamlit, and Hugging Face deployment.

Open Live Demo Back to Portfolio

Project Overview

This project is a computer vision platform that classifies plant leaf images into 15 disease and healthy classes using models trained on the PlantVillage dataset. The goal was to build not only a high-performing model, but also a complete ML workflow from training and evaluation to public deployment.

Methods

I developed and compared a CNN-based ResNet-18 model and a transformer-based Vision Transformer model. Both models used pretrained weights and were adapted for the 15-class crop disease diagnosis task.

  • Resized and preprocessed leaf images for model input.
  • Used transfer learning by replacing ImageNet classification heads.
  • Fine-tuned pretrained model layers for crop disease-specific features.
  • Compared CNN and transformer-based architectures.
  • Evaluated models using accuracy, precision, recall, F1-score, and confusion matrices.

Results

98.2%

ResNet-18 achieved the strongest overall performance, while ViT-B/16 provided a useful transformer-based comparison. The results show that transfer learning can be highly effective for plant disease classification when combined with careful evaluation and deployment.

Deployment

The trained models were deployed through an interactive Streamlit web application. Users can upload a leaf image and view the predicted plant type, disease class, confidence score, and top predictions.

  • Frontend: Streamlit
  • Model hosting: Hugging Face
  • Website integration: GitHub Pages portfolio

Crop Disease Diagnosis Platform

This project is an end-to-end computer vision system for crop disease diagnosis. The system classifies plant leaf images into 15 disease and healthy classes using deep learning models trained on the PlantVillage dataset.

Project Overview

  • Dataset: PlantVillage
  • Total images: 20,638
  • Number of classes: 15
  • Supported plants: Tomato, Potato, and Pepper
  • Task: Multi-class image classification

Methods

I developed and compared two deep learning models: a CNN-based ResNet-18 model and a transformer-based Vision Transformer (ViT-B/16). Both models were initialized with pretrained weights and adapted to the 15-class crop disease classification task.

  • Applied image resizing and tensor preprocessing for model input.
  • Used transfer learning by replacing the original ImageNet classification heads.
  • Fine-tuned deeper model layers to adapt pretrained features to crop disease patterns.
  • Evaluated performance using accuracy, precision, recall, F1-score, and confusion matrices.

Results

  • ResNet-18 test accuracy: 98.2%
  • ViT-B/16 test accuracy: 97.2%
  • ResNet-18 achieved the strongest overall performance.
  • ViT-B/16 provided a useful transformer-based comparison model.

Deployment

The trained models were deployed through an interactive Streamlit web application. Users can upload a leaf image and compare predictions from both models, including predicted plant type, disease class, confidence scores, and top-3 predictions.

  • Frontend demo: Streamlit
  • Model hosting: Hugging Face
  • Website integration: GitHub Pages portfolio
